3. Use Case List

본 시스템에서 사용자는 루프 기반 음악을 생성하고 관리하기 위해 다양한 기능을 사용할 수 있다. 주요 사용 사례는 다음과 같다.

| Use Case           | Actor | Description                                                     |
|--------------------|-------|-----------------------------------------------------------------|
| 회원가입            | User  | 사용자가 계정을 생성하여 시스템을 이용할 수 있도록 한다        |
| 로그인              | User  | 사용자가 자신의 계정으로 로그인하여 저장된 작업 기록을 불러온다 |
| 사운드 재생         | User  | 사운드 라이브러리의 사운드를 재생하여 루프 제작에 활용한다      |
| 마이크 입력 활성화   | User  | 사용자가 마이크 사용 권한을 허용하고 사운드를 녹음할 수 있도록 한다 |
| 루프 생성           | User  | 사용자가 사운드를 녹음하거나 반복하여 루프를 생성한다           |
| 루프 재생 제어       | User  | 생성된 루프의 시작, 정지 등 재생을 제어한다                     |
| 루프 삭제             | User  | 사용자가 마음에 들지 않는 트랙 또는 루프를 개별적으로 삭제한다   |
| BPM 설정           | User  | 사용자가 루프의 템포(BPM)를 설정하거나 변경한다                 |
| 효과 적용          | User  | 루프에 다양한 오디오 효과(Reverb, EQ 등)를 적용한다             |
| 루프 프로젝트 저장   | User  | 사용자가 생성한 루프 프로젝트를 저장한다                        |
| 이전 루프 조회      | User  | 로그인한 사용자가 이전에 저장한 루프 프로젝트 목록을 확인한다   |
| 루프 프로젝트 불러오기| User  | 저장된 루프 프로젝트를 불러와 작업을 이어서 진행한다            |
| 설정 파일 내보내기  | User  | 루프 프로젝트의 BPM, 트랙 구성, 효과 설정값 등을 파일로 저장한다 |
| 설정 파일 업로드    | User  | 다른 사용자의 설정 파일을 업로드하여 동일한 루프 구성을 불러온다 |
