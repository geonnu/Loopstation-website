6. Glossary

|           용어               |                                    설명                               |
| 루프스테이션 (Loop Station) | 사용자가 녹음한 소리를 반복 재생하며 여러 소리를 겹쳐 음악을 만들 수 있는 프로그램이다. |
| 트랙 (Track)               | 녹음된 오디오가 저장되고 재생되는 하나의 단위이다. |
| 루프 (Loop)                | 녹음된 오디오를 일정 구간 단위로 반복 재생하는 기능이다. |
| BPM (Beats Per Minute)   | 음악의 템포를 나타내는 값으로, 1분 동안 반복되는 비트의 수를 의미한다. |
| 브라우저 (Browser)          | 사용자가 프로그램을 실행하고 사용할 수 있는 웹 환경이다. |
| 마이크 권한 (Microphone Permission) | 브라우저에서 사용자의 마이크 사용을 허용하는 권한이다. |
| 트랙 동기화 (Track Synchronization) | 여러 트랙의 루프가 동일한 BPM 기준으로 맞춰 재생되는 기능이다. |
