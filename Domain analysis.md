Domain analysis

1) User :
Guest와 User Actor의 정보를 관리하는 클래스이다.
회원가입 및 로그인 기능과 관련된 사용자 정보를 저장한다.
또한 사용자가 생성한 루프 프로젝트 목록을 관리한다.

2) LoopProject :
사용자가 생성한 루프 프로젝트를 관리하는 클래스이다.
프로젝트 내부에는 BPM 정보, 트랙 정보, 효과 설정값 등이 포함된다.

3) Track :
LoopProject 내부의 개별 루프 트랙을 관리하는 클래스이다.
각 트랙은 녹음된 오디오 데이터를 저장하며 재생 및 삭제 기능과 연결된다.

4) AudioClip :
마이크를 통해 녹음된 오디오 데이터를 관리하는 클래스이다.
녹음 시간과 오디오 데이터 정보를 저장한다.

5) BPMSetting :
루프의 BPM 정보를 관리하는 클래스이다.
여러 트랙이 동일한 BPM 기준으로 동기화되도록 설정한다.

6) AudioEffect :
Reverb, EQ 등의 오디오 효과 정보를 관리하는 클래스이다.
각 트랙 또는 프로젝트에 적용되는 효과 값을 저장한다.

7) SettingFile :
루프 프로젝트 설정 파일을 관리하는 클래스이다.
프로젝트의 BPM, 트랙 구성, 효과 설정값 등을 파일 형태로 저장하거나 불러오는 기능과 연결된다.

8) Server :
사용자의 프로젝트 데이터와 설정 정보를 저장하고 관리하는 클래스이다.
로그인 정보 확인 및 프로젝트 저장 기능을 수행한다.
